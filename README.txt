# CMEE17
Code for CMEE Masters Thesis

CMEE17$ 

├── adj_matrix.py         <-  Cleans video data 
├── combine_data.R        <-  Combines social values from video and RFID data
├── format_iqbal.py       <-  Formats RFID data to match video data 
├── format_vid.R          <-  Format vid data with sparrow pedigree information
├── hpc_a_compare.R       <-  Compares the output of hpc heritability analysis for the age run
├── hpc_compare.R         <-  Compares the output of hpc heritability analysis
├── hpc_run               
│   ├── hered_a_hpc.R     <-  HPC heritability run for age 
│   ├── hered_p_hpc.R     <-  HPC heritability run for base model and density model
│   ├── null_mod2.R       <-  Null model with same sources of variance as the heritability model
│   ├── null_mod.R        <-  Simple null model
│   ├── run_hpc2.sh       <-  Bash script to run analysis on HPC
├── iqbal_hered_test.R    <-  Initial analysis of RFID data
├── null_an.R             <-  Analyse Null model output
├── plot_net_data.R       <-  Makes plots from the network data outputs
├── rep_alf.R             <-  Male vs. Female repeatabilities  
├── sparrow_cage2.py      <-  A general use file to make interaction times from RFID data
├── sparrow_cage_iqbal.py <-  Makes an output of interactions from Raw RFID data
├── vid_hered.R           <-  Make networks and get social data from video data
